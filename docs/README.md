# Documentation Index

This folder contains the current project documentation.

## Files

- `ui-guide.md`
  Describes the user interface: top-level actions, scene controls, settings, tables, timeline, and statistics.

- `optimization-pipeline.md`
  Describes the data flow from input task to chromosome, decoding, fitness evaluation, GA generations, baseline comparison, and UI output.

- `variation-and-evolution.md`
  Describes how solutions change over time: inheritance, crossover, mutations, elitism, duplicates, and immigrants.

- `task-format.md`
  Describes the `.txt` task format used by the loader.

## Scope

Old architecture notes and one-off decision logs were removed on purpose.
The current documentation is organized around how the program is used and how it works today.
